# Сайт-визитка: Школа №12 г. Керчь

## Готовые файлы сайта

В репозитории подготовлены три файла, которые нужно загрузить на хостинг:

| Файл | Назначение |
|------|-----------|
| `index.html` | Главная страница сайта |
| `style.css` | Стили (оформление сайта) |
| `school.jpg` | Фотография школы |

---

## Как загрузить файлы на Hostinger (пошагово)

### Шаг 1 — Войти в панель управления

1. Открыть сайт **[hpanel.hostinger.com](https://hpanel.hostinger.com)**
2. Ввести свой **Email** и **Пароль** → нажать **Войти**

> ⚠️ Если не пускает — попробуйте **очистить cookies** браузера:
> Chrome: `Ctrl+Shift+Del` → выбрать «Файлы cookie» → очистить → перезагрузить страницу

---

### Шаг 2 — Открыть Файловый менеджер

1. В панели найти свой хостинг → нажать кнопку **Управление**
2. В левом меню найти раздел **«Файлы»** → нажать **«Файловый менеджер»**
3. Перейти в папку **`public_html`** (двойной клик)

---

### Шаг 3 — Удалить стандартные файлы (если есть)

Если в папке `public_html` уже есть файлы (например `index.php` или `index.html`) — **удалите их**, чтобы они не мешали вашему сайту.

---

### Шаг 4 — Загрузить файлы сайта

1. В Файловом менеджере нажать кнопку **«Загрузить файлы»** (или «Upload Files»)
2. Загрузить все три файла:
   - `index.html`
   - `style.css`
   - `school.jpg`
3. Дождаться завершения загрузки

---

### Шаг 5 — Проверить сайт

1. Открыть браузер и перейти на ваш домен (например: `https://ваш-домен.ru`)
2. Сайт должен открыться и отображаться корректно

---

## Скачать файлы с GitHub

Чтобы скачать файлы на компьютер с GitHub:

1. Перейти на страницу репозитория: **[github.com/ggYODA/shool12](https://github.com/ggYODA/shool12)**
2. Нажать зелёную кнопку **`<> Code`**
3. Выбрать **`Download ZIP`**
4. Распаковать архив на компьютере
5. Загрузить три файла (`index.html`, `style.css`, `school.jpg`) на хостинг
